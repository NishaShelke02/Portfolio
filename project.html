SwachhSeva – Public Sanitation Complaint System

SwachhSeva is a full-stack web application built to improve urban sanitation through active citizen participation and efficient municipal management. The platform allows citizens to register complaints regarding issues like garbage accumulation, clogged drains, or unclean streets by uploading details, photos, and location data.

Key Features:

  • Role-based authentication (RBAC) for citizens, volunteers, supervisors, and managers.
  • Real-time complaint tracking and resolution workflows.
  • Interactive dashboards and performance monitoring with Chart.js.
Tech Stack: React.js, Bootstrap, .NET, Spring Boot, MySQL, Chart.js

Impact: Fosters transparency, accountability, and effective public sanitation management.

Food Rescue Web Application

A platform designed to reduce food waste by connecting surplus food donors (restaurants, households, events) with NGOs and individuals in need. It enables real-time food donations, efficient distribution, and transparency through role-based dashboards.

Key Features:

  • Donor Dashboard: Add surplus food details and track donation status.
  • NGO Dashboard: View available food, claim donations, and manage distribution.
  • Real-time updates on available and claimed food.
  • APIs with JSON for structured, secure communication.
Tech Stack: React.js, Node.js, Express.js, MongoDB/MySQL, Bootstrap

Impact: Minimizes food waste and supports NGOs in timely distribution.

Pet Adoption Management System

A web-based platform that connects pet shelters with adopters, streamlining the adoption process through a secure and user-friendly interface.

Key Features:

  • User Authentication with JWT for secure role-based access.
  • Pet Listings: Add, update, and manage pet profiles with images.
  • Adoption Applications: Submit and track adoption requests online.
  • Admin Dashboard for user and application management.
Tech Stack: React.js, Spring Boot, MySQL, JWT, Bootstrap

Impact: Simplifies adoption, improves shelter management, and promotes responsible pet ownership.

Plant Nursery Management System

A web application designed to digitalize nursery operations by managing plant inventory, sales, and customer interactions in a seamless way.

Key Features:

  • Inventory Management: Track and update plant stock and availability.
  • Order & Billing System: Customers can place orders with automatic bill generation.
  • Customer Portal: Browse plants, view details, and order online.
  • Admin Dashboard for sales reports and customer interaction monitoring.
Tech Stack: ASP.NET Core (.NET), React.js, MySQL, Bootstrap

Impact: Reduces manual work, ensures accurate stock management, and enhances customer experience.
